
Quero criar um sistema web agêntico (MCP+MoE+LLM) para rodar localmente, similar ao Trae Ai, tão bom quanto ele, que possa codar, acessar arquivos localmente. Será meu orquestrador, engenheiro sênior e CTO local. Rodará em uma interface web, mas poderá usar o VS Code, navegadores, serviços e protocolos via terminal (inclusive cmd e power shell) para desenvolvimento, sem restrições, capaz de produzir os artefatos necessários para cada projeto. 

O agente poderá melhorar sua própria interface, criar customizações e novas ferramentas para seu uso, de acordo com a necessidade dos projetos. Está em uma máquina isolada, dedicada para ele. Só não está autorizado a contratar serviços que impliquem em custo ou risco legal. Nenhum código deve ser enviado "para fora". Preferencialmente gera localmente. 

Quero que esse sistema possua um painel integrado, que permita conectá-lo a uma LLM local de forma descomplicada, e que também permita o uso, rotação e gerenciamento de chaves de API remotas (gratuitas e pagas) de forma segura. 

Quero que ele use RAG e sistemas de dados locais para aumentar a capacidade dos contextos, armazenamento e histórico de chats.

- Coldfusion (Especialista)
- Git
- GitHub
