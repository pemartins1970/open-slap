# Kickoff (Simples) — Mini App de Lista de Tarefas

## Contexto

Quero construir um mini app web simples para gerenciar tarefas pessoais.

## Objetivo

Criar um MVP funcional com cadastro de tarefas, marcação como concluída e persistência local.

## Escopo (MVP)

- Criar / listar / marcar como concluída / remover tarefas
- Persistência local (LocalStorage) ou SQLite (se fizer sentido no stack escolhido)
- Interface simples, responsiva e legível

## Fora de escopo (agora)

- Login, multiusuário, sincronização em nuvem
- Notificações push
- Integrações externas

## Restrições

- Rodar localmente em Windows
- Sem deploy público

## Entregáveis

- README com instruções de execução
- Backlog inicial (tarefas técnicas)
- Protótipo de UI (estrutura de telas)

## Critérios de aceite

- Consigo criar e concluir tarefas
- Dados persistem ao recarregar
- Projeto roda com 1 comando por lado (backend/frontend), ou apenas frontend se não houver backend
