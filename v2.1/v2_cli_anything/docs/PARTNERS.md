# Parceiros — Open Slap!

Open Slap! é um projeto open source desenvolvido de forma independente.  
Não há parceiros comerciais formais nesta versão. Essa seção documenta o que valorizamos em uma eventual parceria e como o ecossistema se relaciona.

---

## Ecossistema de produtos

### Slap! PRO *(em desenvolvimento)*

O Slap! PRO é o produto comercial derivado do Open Slap!. Herda o motor agêntico deste projeto e adiciona:
- Autenticação OAuth com providers externos
- Conectores proprietários
- Supervisão avançada de agentes
- SLA e suporte técnico
- Interface para usuário final não-técnico

A existência do Slap! PRO financia a continuidade do Open Slap! como projeto open source.

---

As seguintes plataformas são suportadas nativamente como providers de LLM:

| Provider | Tipo | Link |
|----------|------|------|
| Google Gemini | API / Nuvem | [aistudio.google.com](https://aistudio.google.com) |
| Groq | API / Nuvem | [console.groq.com](https://console.groq.com) |
| OpenAI | API / Nuvem | [platform.openai.com](https://platform.openai.com) |

A integração é de código aberto e não envolve acordo comercial com nenhum desses providers.

## Interesse em parcerias?

Se você representa uma organização — especialmente escola pública, universidade, ONG de educação ou coletivo de makers — e quer usar o Open Slap! em projetos educacionais ou de pesquisa, entre em contato.

Parcerias de uso educacional são bem-vindas e podem envolver:
- Suporte informal no setup inicial
- Inclusão no README como caso de uso
- Colaboração em documentação ou conteúdo pedagógico

Não oferecemos SLA, suporte técnico continuado ou personalizações pagas no contexto do projeto open source.

# Partners — Open Slap!

Open Slap! is an independently developed open source project.  
There are no formal commercial partners in this version. This section documents what we value in a potential partnership and how the ecosystem relates.

---

## Product ecosystem

### Slap! PRO *(in development)*

Slap! PRO is the commercial product derived from Open Slap!. It inherits the agentic engine from this project and adds:
- OAuth authentication with external identity providers
- Proprietary connectors
- Advanced agent supervision
- SLA and technical support
- Interface for non-technical end users

The existence of Slap! PRO funds the continuity of Open Slap! as an open-source project.

## Interested in a partnership?

If you represent an organisation — especially a public school, university, education NGO, or makers collective — and want to use Open Slap! in educational or research projects, get in touch.

Educational use partnerships are welcome and may involve:
- Informal support on initial setup
- Inclusion in the README as a use case
- Collaboration on documentation or pedagogical content

We do not offer SLA, ongoing technical support, or paid customisations in the context of the open-source project.
